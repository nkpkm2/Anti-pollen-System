# Anti-pollen System

A multimodal AI-assisted system for pollen-related risk estimation and environmental response.

## What this project does
This system integrates multiple weak signals such as particle sensing, light response, weather data, time, and user feedback to estimate pollen-related risk and support better daily decisions.

## Why this matters
Real-time pollen detection is difficult, slow, and expensive. Instead of relying on precise measurement, this project focuses on multimodal inference and practical decision support.

## Core idea
- Multimodal sensing instead of single precise measurement
- State estimation instead of direct observation
- Decision support instead of simple automation
- Human-in-the-loop feedback
- Closed-loop environmental response
