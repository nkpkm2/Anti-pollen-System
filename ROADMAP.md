# Roadmap

## Phase 0: Project framing
- Define project scope
- Build repository structure
- Establish documentation system

## Phase 1: Sensing prototype
- Validate light-based sensing routes
- Measure signal separability
- Design ADC-compatible voltage shifting

## Phase 2: Data pipeline
- Collect and store multimodal observations
- Build simple backend and dashboard

## Phase 3: Baseline inference
- Build first risk estimation logic
- Combine sensor data, weather, time, and feedback

## Phase 4: Personalized AI
- Incorporate user feedback
- Improve risk estimation with historical data

## Phase 5: Closed-loop response
- Connect environmental control
- Deliver proactive suggestions and actions
