# Problem Definition

## Core Problem
Pollen-related discomfort is real and impactful, but direct real-time pollen sensing is difficult to implement in a low-cost personal system.

## Engineering Challenge
No single signal is sufficient:
- particle count is not specific
- optical response is incomplete
- weather is indirect
- user discomfort is subjective

## Project Hypothesis
A useful system can be built by combining multiple weak signals and estimating pollen-related risk instead of directly measuring pollen concentration.

## Practical Objective
Support better daily decisions such as:
- whether to go outside
- how to adjust purifier settings
- when to expect higher environmental risk
