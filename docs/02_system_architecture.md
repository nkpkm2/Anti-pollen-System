# System Architecture

## High-Level Layers
1. Sensing
2. Feature extraction
3. State estimation
4. Decision support
5. Feedback loop

## Sensing Layer
Possible sources:
- light-based sensing
- particle-related sensing
- weather API
- time and calendar context
- user feedback

## State Estimation Layer
Estimate:
- pollen-related risk
- environmental discomfort risk
- opportunity windows for better decisions

## Output Layer
The system may provide:
- purifier control suggestions
- behavioral suggestions
- future risk reminders
