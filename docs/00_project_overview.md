# Project Overview

## Goal
Build a practical multimodal system to estimate pollen-related risk and support better environmental and behavioral decisions.

## Problem
Direct real-time pollen measurement is difficult and expensive. In everyday settings, a more practical approach is to combine multiple indirect signals and infer risk instead of trying to measure pollen concentration precisely.

## Approach
The system uses multiple weak but complementary signals, including:
- optical sensing
- particle-related signals
- weather data
- time and seasonal context
- user feedback

These inputs are combined to estimate a pollen-related risk level.

## System Structure
- sensing
- feature extraction
- state estimation
- decision support
- feedback loop

## First Version Focus
- validate sensing methods
- build signal conditioning path
- define multimodal input structure
- prepare baseline inference logic
