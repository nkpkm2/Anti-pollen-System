# State Estimation and Decision Logic

## Why State Estimation Is Needed
Important user-relevant states cannot be measured directly. The system therefore estimates them from multiple imperfect signals.

## Candidate Estimated States
- pollen-related risk
- discomfort likelihood
- environmental suitability
- opportunity for preventive action

## Decision Logic
The system should not only report sensor values. It should transform multimodal inputs into practical decisions and suggestions.

## Design Principle
Estimate state first, then decide action.
